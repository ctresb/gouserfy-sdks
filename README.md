# Gouserfy SDKs

Official client libraries for [Gouserfy](https://github.com/ctresb/gouserfy) - A simple and straightforward user management backend.

## Available SDKs

| Language | Path | Status |
|----------|------|--------|
| TypeScript | [typescript/](typescript/) | Available |

## Usage

Each SDK has its own README with installation and usage instructions.

## Backend

These SDKs are designed to work with the Gouserfy backend. See the [main repository](https://github.com/ctresb/gouserfy) for setup instructions.
